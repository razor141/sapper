# sapper
About the Project

This is a simple Minesweeper game written in Python.
The goal of the game is to open all safe cells without hitting a mine.

Features

Random mine generation

Simple game logic

Win and lose conditions

Console interface

How to Run
Run Python version

Install Python

Run the file:

python main.py

Run EXE version

Run the compiled file:

dist/minesweeper.exe
